O Git Bash é o aplicativo para ambientes do Microsoft Windows que oferece a camada de emulação 
para a experiência de linha de comando Git. Bash é acrônico para "Bourne Again Shell".

O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS.
 Ele inclui suporte para depuração, controle de versionamento Git incorporado, realce de sintaxe, complementação inteligente
  de código, snippets e refatoração de código.
